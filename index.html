So what does handlebars do?

Handlebars is a JavaScript templating system that lets you write less HTML and stop repeating yourself!

Handlebars uses moustache templates

Insert your content {{here}}!

Who is it for?

Anyone who has to display a lot of data, or data that changes or is an unpredictable amount of data.

Ex. blog

Template:

JavaScript:

        
          var templateSource = document.getElementById('template').innerHTML;
          var template = Handlebars.compile(templateSource);
          var resultsPlaceholder = document.getElementById('-results');
          var data = {
            "game": "Papers please",
            "developer": "Lucas Pope"
          };
          resultsPlaceholder.innerHTML = template(data);
